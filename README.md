基于GitHub Pages部署，无需后端服务器- 📱 **响应式设计** - 完美适配PC、平板、手机等各种设备- 📊 **交互式图表** - 使用Chart.js提供流畅的数据可视化体验- 🔄 **灵活数据源** - 支持JSON、CSV格式，可配置自定义数据URL- 🎨 **现代化UI** - 采用紫色渐变配色方案，美观大方- ⚡ **高性能** - 优化加载速度，首屏加载<2秒- 🛡️ **错误处理** - 完善的CORS和数据验证机制---## 🚀 主要功能### 1. 数据可视化- **交互式折线图**：展示近15天的TTM股息率走势- **悬浮提示**：鼠标悬停显示具体数值- **响应式图表**：自动适配不同屏幕尺寸### 2. 数据表格- **动态表格**：展示日期和股息率详细数据- **格式化显示**：日期格式化（YYYY-MM-DD）、百分比格式（2位小数）- **清晰布局**：条纹背景，易于阅读### 3. 数据源配置- **自定义URL**：支持输入任意公开API地址- **多格式支持**：自动识别JSON和CSV格式- **演示模式**：内置演示数据，方便快速测试### 4. 错误处理- **CORS检测**：自动检测跨域问题并提供解决建议- **数据验证**：验证数据格式，提示错误原因- **友好提示**：用户友好的错误消息---## 🎬 快速开始### 在线访问直接访问在线演示：[https://wdsdwsh1135-del.github.io/dividend-low-volatility-analysis/](https://wdsdwsh1135-del.github.io/dividend-low-volatility-analysis/)### 本地运行1. **克隆仓库**```bashgit clone https://github.com/wdsdwsh1135-del/dividend-low-volatility-analysis.gitcd dividend-low-volatility-analysis```2. **启动本地服务器**使用Python 3:```bashpython -m http.server 8000```或使用Node.js的http-server:```bashnpx http-server -p 8000```3. **打开浏览器**访问 `http://localhost:8000`---## 📖 使用说明### 加载演示数据1. 打开网站2. 点击 **"加载演示数据"** 按钮3. 系统会生成近15天的模拟股息率数据4. 查看图表和表格展示### 加载自定义数据1. 准备符合格式要求的数据文件（JSON或CSV）2. 将数据文件发布到公开URL（需支持CORS）3. 在输入框中填入数据URL4. 点击 **"加载数据"** 按钮### 数据源示例**推荐的公开数据源**：- GitHub Gist（支持CORS）- JSONPlaceholder- 自建支持CORS的API---## 📄 数据格式### JSON格式```json{  "data": [    {      "date": "2026-02-03",      "yield": 3.45    },    {      "date": "2026-02-04",      "yield": 3.52    }  ]}```或简化格式：```json[  { "date": "2026-02-03", "yield": 3.45 },  { "date": "2026-02-04", "yield": 3.52 }]```### CSV格式```csvdate,yield2026-02-03,3.452026-02-04,3.522026-02-05,3.48```**字段说明**：- `date`：日期，格式为 YYYY-MM-DD- `yield`：股息率（TTM），数值类型，单位为百分比---## 🛠️ 技术栈| 技术 | 版本 | 用途 ||------|------|------|| HTML5 | -
